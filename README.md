# SAKTI - Sistem Akuntansi & Keuangan Terintegrasi (SMK Akbar)

**SAKTI** adalah aplikasi berbasis web yang dirancang untuk mengelola sistem pembayaran SPP dan manajemen keuangan di SMK Akbar. Project ini bertujuan untuk mendigitalisasi pencatatan keuangan agar lebih transparan, akurat, dan efisien.

## 🚀 Fitur Utama
- **Manajemen Data Siswa:** Kelola informasi siswa dan status pembayaran.
- **Pembayaran SPP:** Pencatatan transaksi SPP secara real-time.
- **Laporan Keuangan:** Rekapitulasi pemasukan dan pengeluaran sekolah.
- **Multi-user:** Akses yang berbeda untuk Admin dan Petugas Keuangan.
- **Mode Offline & Online:** Dirancang untuk fleksibilitas akses di lingkungan sekolah.

## 🛠️ Teknologi yang Digunakan
- **Bahasa Pemrograman:** PHP
- **Framework:** Laravel Query Builder
- **Database:** MySQL
- **Frontend:** Bootstrap, HTML5, CSS3, JavaScript
- **Alat Bantu:** XAMPP / Laragon

## 📋 Prasyarat
Sebelum menjalankan project ini, pastikan kamu sudah menginstal:
- PHP >= 8.x
- Composer
- MySQL/MariaDB
